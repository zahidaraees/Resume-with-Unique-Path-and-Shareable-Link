## How to Take Screenshots

### Option 1: Browser DevTools (Recommended)
1. Open your app in Chrome/Edge
2. Press `F12` to open DevTools
3. Press `Ctrl+Shift+M` (Windows) or `Cmd+Shift+M` (Mac) to toggle device mode
4. Select your device size
5. Press `Ctrl+Shift+P` → Type "screenshot" → Select "Capture full size screenshot"

### Option 2: Browser Extensions
- Use extensions like **GoFullPage** for full-page screenshots
- Available for Chrome, Firefox, and Edge

### Option 3: Online Tools
- Use [screenshot.guru](https://screenshot.guru) for high-quality website screenshots
- Or [screenshotapi.net](https://screenshotapi.net) for programmatic screenshots
